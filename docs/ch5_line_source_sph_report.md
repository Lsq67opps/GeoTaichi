# 第5章二维线源瞬时抛泥算例：SPH 原始问题整理

依据《论文翻译.md》第5章内容，汇总二维线源瞬时抛泥（静水下抛泥）问题的物理模型、控制方程以及边界/初始条件与数值处理，按图示四个问题作答。

## 1. 物理类别
- **流体类型**：水–泥沙双流体，弱可压缩（通过状态方程控制压缩性）。  
- **颗粒材料**：泥沙颗粒，给定粒径 \(d_p\) 与单颗粒沉速 \(\omega_s\)（见表5.1）。  
- **自由表面/冲击/多相**：存在自由水面，SPH 直接捕捉；水相与沙相互溶型双流体（体积分数叠加为1），无相变，重力主导。

## 2. 控制方程（SPH 背后的 PDE）
对每一相 \(k=f,s\)：  
- **质量守恒**  
  \[
  \frac{\partial(\alpha_k \rho_k)}{\partial t} + \nabla \cdot (\alpha_k \rho_k \mathbf{u}_k)=0
  \]
- **动量守恒**  
  \[
  \frac{\partial(\alpha_k \rho_k \mathbf{u}_k)}{\partial t}
  + \nabla\cdot(\alpha_k \rho_k \mathbf{u}_k \mathbf{u}_k)
  = -\alpha_k \nabla p + \nabla\cdot(\alpha_k \boldsymbol{\tau}_k) + \alpha_k \rho_k \mathbf{g} + \mathbf{F}_{A k}
  \]
  其中 \(\mathbf{F}_{A k}\) 为相间作用力（拖曳等）；泥沙分压忽略但受同一压力场作用。  
- **状态方程**：水相压力由 Tait 类状态方程给出以保持弱可压缩；泥沙压强忽略。  
- **湍流/扩散**：SPS Smagorinsky 次滤波应力，系数 \(C_s=0.1\)；泥沙施密特数 \(S_c=1.0\) 控制标量扩散。

## 3. 边界条件 & 初始条件
- **几何/边界**：二维矩形域，宽度 \(L=1\,\mathrm{m}\)，水深 \(H=1\,\mathrm{m}\)。侧壁与底部为固壁，采用固定不动的三层 SPH 边界粒子；顶部为自由水面，无显式条件。  
- **初始场**：静水，速度为零，水面平静。自由水面附近瞬时释放泥沙斑块，初始面积 \(q_0 \in \{5,10\}\,\mathrm{cm}^2\)，初始体积分数 \(\alpha_{s0}=0.606\)，初始速度为零。  
- **颗粒参数**：六组实验（08C1/13C1/50C1，08C2/13C2/50C2）分别给定 \(d_p, \omega_s, q_0\)。  
- **离散尺度**：计算采用粒径 \(\Delta=0.0050\,\mathrm{m}\) 的 SPH 粒子离散水体与泥沙。

## 4. SPH 中的特殊处理/物理
- **自由表面捕捉**：无网格拉格朗日粒子自然形成自由表面。  
- **数值稳定性**：人工黏性、Shepard 密度平滑（式(4.120) 修正）抑制波动；时间步长满足 CFL/加速度/黏性约束。  
- **相间耦合**：体积分数法，压强共享，相间拖曳显式进入动量方程。  
- **忽略项**：未考虑表面张力；无入流/出流和相变，仅重力外力。
